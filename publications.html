Publications

IoT-Based Real-Time Patient Monitoring System with Predictive Analytics

MD. Rakib Hassan Dipu, Co-authors

IEEE International Conference on Recent Advances in Automation and Industrial Control (RAAICON)

This research presents a low-cost IoT-based healthcare monitoring system integrating ESP32, biomedical sensors, and machine learning models for early detection of critical patient conditions. The system demonstrates improved response time and reduced false alarms compared to traditional threshold-based monitoring.

  • ESP32-based biomedical sensing
  • MQTT cloud communication
  • Machine learning anomaly detection
  • Healthcare IoT automation
