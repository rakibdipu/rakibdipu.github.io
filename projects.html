Lightweight IoT-Driven Panic Attack Detection & Caregiver Alert System

Healthcare IoT ESP32 Machine Learning IEEE RAAICON 2025

Problem Statement

Individuals with Autism Spectrum Disorder (ASD) and anxiety disorders often experience sudden panic attacks that require immediate caregiver intervention. Existing monitoring solutions are either reactive, rely on manual observation, or are prohibitively expensive for large-scale deployment in low-resource settings.

Solution & Technical Approach

I designed a lightweight, low-cost IoT system capable of continuously monitoring physiological and environmental parameters using an ESP32-based embedded platform. Sensor data is processed locally and transmitted via MQTT to a cloud dashboard for real-time visualization and alerting.

A machine learning pipeline was developed using Python and scikit-learn to detect early indicators of panic attacks. The trained model was optimized for deployment on resource-constrained devices, enabling near-real-time prediction before critical escalation.

My Role: Lead system architect and first author. Responsible for hardware selection, PCB prototyping, embedded firmware development, ML model design, data analysis, and system validation.

System Demonstration

Complete walkthrough covering sensor setup, embedded processing, live dashboard monitoring, and caregiver alert mechanism.

Technology Stack

Hardware: ESP32, physiological sensors, custom PCB
Firmware: C++, Arduino framework
ML: Python, scikit-learn, Random Forest
Communication: MQTT, Wi-Fi
Backend: Node-RED, cloud dashboard
Tools: EasyEDA, Git, VS Code

Adaptive IoT Environment for Autism Care

Assistive Technology IoT Sensors

Problem Statement

Individuals with autism often experience sensory overload due to uncontrolled environmental conditions such as light, temperature, and noise, negatively affecting comfort and behavior.

Solution & Technical Approach

Developed an adaptive IoT-based environment monitoring and control system that dynamically adjusts environmental parameters using sensor feedback. The system enables caregivers to monitor conditions remotely and automate responses based on predefined thresholds.

My Role: System designer and embedded developer. Responsible for sensor integration, ESP32 firmware, and system testing.

Results & Impact

  • Improved environmental stability in controlled test environments
  • Reduced manual caregiver intervention
  • Demonstrated feasibility for assistive smart environments
